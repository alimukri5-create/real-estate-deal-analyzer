# System Prompts & Instructions

## Deal Analyzer — v1 Prompts

### Extraction Prompt
"Extract all property facts from the following memo text. Focus on: address, price, bedrooms, sqft, rent, yield, tenure, EPC rating, council tax band, and any key bullet points."

### Valuation Prompt
"Given these facts and a BTL yield strategy, compute fair value bands (low/base/high) and flag as overpay/fair/underpay."

### Memo Prompt
"Generate a structured markdown investment memo with: extracted facts, valuation table, scenario analysis, verdict, and open questions."
